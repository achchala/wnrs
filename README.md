# wnrs <3 (virtual edition)

iOS app and website  for playing We’re Not Really Strangers (three levels, wildcards, dig-deeper prompts, and a short closing card). also has the option to include the Honest Dating expansion pack.

## run it

1. open `WNRS.xcodeproj` in Xcode
2. pick your iPhone or a simulator
3. run (⌘R).
